# Web3-React-fronted

## 创建项目
pnpm create vite Web3-React-fronted --template react-ts

## 项目技术栈
- React + TypeScript + Vite
- 路由：React Router v6  **[官方文档：https://reactrouter.com/start/data/installation]**
- UI 组件库：shadcn/ui **[官方文档：https://ui.shadcn.com/docs]**
- TailwindCSS **[官方文档：https://tailwindcss.com/docs/installation/using-vite]**
